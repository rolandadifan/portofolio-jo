Project

Belajar Santuy Design

Belajar Skuy, Design

Affinity mapping dilakukan dapat membantu dalam mengelompokkan informasi, fakta, riset etnografi, ide dari brainstorm, pendapat user, kebutuhan pengguna, insight, masalah desain, dan lainnya. Serta untuk membantu mencari problem yang sangat sering mencul ketika sehingga dapat menjalin komunikasi yang baik antara stakeholder dan designer

Sitemap bertujuan untuk melihat seberapa besar cakupannya dan juga sebagai panduan saya saat mendesain halaman. Setelah sitemap selesai, saya melanjutkan ke langkah berikutnya, wireframe.

Setelah selesai membuat sitemap saya melanjutkan pada proses wireframe. Saya membuat wireframe sedikit lebih simple agar nantinya saat pembuatan mockup dan diimplementasikan kepada app user lebih mudah dan gampang mengerti menggunakan applikasi ini.

Sebelum menuju ke mockup, saya melakukan riset warna apa yang harus saya gunakan. Saya juga membandingkannya dengan pesaing. Selain warna, saya juga memerlukan beberapadesain apa yang harus saya gunakan untuk halaman, dan saya menemukan ini.

Saya juga menambahkan beberapa pop up ketika berhasil atau gagal melakukan aksi

Setelah menyelesaikan setiap tahapan dan memirkan kan ulang kembali tahapan. Saya memutuskan untuk melanjutkan untuk membuat Mockup dari app Belajar Skuy. Dengan mengisikan gambar, memberi warna dan menyempurnakan tata letak.

Deskripsi

Target : User yang ingin mengikuti Course online
Tahun : 2020

Belajar Skuy merupakan salah satu platform app mobile yang memudahkan seseorang untuk mencari course dibidang UI designer, UX Designer, UX Research, dan UI/UX Designer. Selain ituBelajar Skuy memiliki materi yang mudah untuk dipahami dan metode belajar yang kreatif dan inovatif.
